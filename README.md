## AI Assistant for those in need

---

An easy-to-use ChatGPT-like AI Assistant.

One can run this little websitey program by running the following commands:
```bash
npm i
npx tsc -p ./tsconfig.json
```

These commands will compile the TypeScript code into JavaScript and install the necessary dependencies.
If one wants to use this assistant, one must first acquire an API key at [OpenAI](https://platform.openai.com/signup).
This sadly does require you to add some money to your account, but one must consider it to be worth it.

The purpose of this program is to provide one with a very useful AI tool that will be able
to execute a variety of tasks, such as:

- Respond to your questions
- Provide you with information
- Be up to date with actual information past the cut-off date of the model
- Provide you with a variety of tools to help you with your daily tasks

Some of these features are yet to be implemented, so stay tuned for updates,
once I found a way to download more time into my day.
